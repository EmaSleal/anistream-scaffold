# Rutina de Ejercicios Organizada

## Estructura Semanal (6 Sesiones PPL A/B)

| Sesión | Tipo | Grupos Musculares | Duración Estimada |
|--------|------|-------------------|------------------|
| Pull A | Tirón A | Espalda (grosor) + Bíceps | ~75 min |
| Push A | Empuje A | Pecho + Tríceps + Hombro | ~75 min |
| Leg A | Piernas A | Cuádriceps + Pantorrilla + Core | ~60 min |
| Pull B | Tirón B | Espalda (amplitud) + Bíceps | ~75 min |
| Push B | Empuje B | Hombro + Pecho accesorios + Tríceps | ~75 min |
| Leg B | Piernas B | Glúteo + Isquios + Pantorrilla + Core | ~60 min |

---

## PULL A — Espalda (Grosor) + Bíceps

### Ejercicios Clave

**Compuestos:**
1. Remo con barra T — 4 sets x 6–8 reps (RIR 1–2)
2. Remo pecho apoyado — 3 sets x 8–10 reps (RIR 2)
3. Remo en polea baja — 3 sets x 10–12 reps (RIR 2)

**Bíceps:**
4. Curl barra Z — 3 sets x 8–10 reps (RIR 1–2)
5. Curl inclinado — 3 sets x 10–12 reps (RIR 2)

**Accesorios:**
6. Curl en polea — 2 sets x 12–15 reps (RIR 3)
7. Face pulls — 2 sets x 15–20 reps (salud escapular)

**KPI de Progresión:** Remo barra T

---

## PUSH A — Pecho + Tríceps + Hombro

### Ejercicios Clave

**Pecho:**
1. Press banca con barra — 4 sets x 6–8 reps (RIR 1–2)
2. Press inclinado con mancuernas — 3 sets x 8–10 reps (RIR 2)
3. Fondos en paralelas — 3 sets x 8–10 reps (RIR 2)

**Tríceps:**
4. Press cerrado — 3 sets x 8–10 reps (RIR 1–2)
5. Extensión en polea — 3 sets x 12–15 reps (RIR 2)

**Hombro:**
6. Press militar con mancuernas — 2 sets x 8–10 reps (RIR 2)

**Accesorios:**
7. Cruce de poleas — 2 sets x 12–15 reps
8. Pec deck — 2 sets x 15–20 reps (finisher)

**KPI de Progresión:** Press banca

---

## LEG A — Cuádriceps + Pantorrilla + Core

### Ejercicios Clave

**Cuádriceps:**
1. Hack squat / Smith talones elevados — 4 sets x 8–10 reps (RIR 1–2)
2. Prensa inclinada — 3 sets x 10–12 reps (RIR 2)
3. Búlgaras — 3 sets x 10–12 reps por lado (RIR 2)

**Aislamiento Quad:**
4. Extensión de rodilla — 3 sets x 12–15 reps (RIR 2–3)
   - Última serie: dropset x 8–10 reps adicionales

**Pantorrilla:**
5. Elevación de pie — 3 sets x 12–15 reps (RIR 2)
6. Elevación sentado — 2 sets x 12–15 reps (RIR 2)

**Core:**
7. Plancha — 2 sets x 45–60 seg
8. Rueda abdominal — 2 sets x 10–15 reps

**KPI de Progresión:** Hack squat / Smith

---

## PULL B — Espalda (Amplitud) + Bíceps

### Ejercicios Clave

**Compuestos:**
1. Dominadas — 4 sets x 6–10 reps (RIR 1–2)
2. Jalón al pecho — 3 sets x 10–12 reps (RIR 2)
3. Pullover en polea — 3 sets x 12–15 reps (RIR 2)

**Bíceps:**
4. Curl predicador — 3 sets x 10–12 reps (RIR 1–2)
5. Curl martillo — 3 sets x 12–15 reps (RIR 2)

**Accesorios:**
6. 21s (curl barra) — 2 sets x 21 reps (metabólico)
7. Face pulls — 2 sets x 15–20 reps

**KPI de Progresión:** Dominadas

---

## PUSH B — Hombro + Pecho Accesorios + Tríceps

### Ejercicios Clave

**Hombro:**
1. Press militar con mancuernas — 4 sets x 8–10 reps (RIR 1–2)
2. Elevaciones laterales — 3 sets x 12–15 reps (RIR 2)
   - Última serie: dropset x 8–10 reps adicionales
3. Elevaciones frontales — 2 sets x 12–15 reps (RIR 2)

**Posterior:**
4. Pájaros — 3 sets x 12–15 reps (RIR 2)

**Trapecio:**
5. Encogimientos — 2 sets x 12–15 reps (RIR 2)

**Pecho Accesorios:**
6. Cruce de poleas — 2 sets x 12–15 reps
7. Push-ups — 2 sets x 15–25 reps (finisher, tempo controlado)

**Tríceps:**
8. Copa con mancuerna (overhead) — 2 sets x 12–15 reps (RIR 2)

**KPI de Progresión:** Press militar

---

## LEG B — Glúteo + Isquios + Pantorrilla + Core

### Ejercicios Clave

**Glúteo:**
1. Hip thrust — 4 sets x 8–10 reps (RIR 1–2, pausa 2–3 seg arriba)
2. Abducción en máquina — 3 sets x 12–15 reps (RIR 2)
3. Pull-through — 3 sets x 10–12 reps (RIR 2)

**Cadena Posterior:**
4. Peso muerto rumano — 3 sets x 8–10 reps (RIR 1–2)

**Isquios:**
5. Curl femoral — 3 sets x 10–12 reps (RIR 2)
6. RDL unilateral — 2 sets x 10–12 reps por lado (RIR 2)

**Pantorrilla:**
7. Elevación de pie — 3 sets x 12–15 reps (RIR 2)
8. Tibialis raise — 2 sets x 15–20 reps (balance articular)

**Core:**
9. Side plank — 2 sets x 45–60 seg por lado
10. Elevaciones de piernas — 2 sets x 12–15 reps

**KPI de Progresión:** Hip thrust

## Regla de Arquitectura (Crítica)

⚠️ **No modificar qué ejercicios van en qué día.**

Solo se ajusta:
- ✓ Carga
- ✓ Repeticiones
- ✓ Series (dentro de rangos específicos)
- ✓ Tempo y RIR

---

## Notas Importantes

- Cada sesión está diseñada para ~2 horas de volumen total
- Los ejercicios clave (KPI) son los que deben progresar prioritariamente
- Los accesorios pueden variar ligeramente según disponibilidad de equipo
- Mantener tempo controlado especialmente en aislamientos (2–3s en negativa)
- No hay variación de ejercicios entre semanas en la misma sesión (solo dentro del ciclo)

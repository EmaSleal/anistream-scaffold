# Dashboard Semanal de Entrenamiento (KPI Framework)

## 1. Resumen Ejecutivo (Vista Rápida)

Sistema de "semáforo" para evaluación semanal rápida.

| KPI | Objetivo | Estado |
|-----|----------|--------|
| Cumplimiento de sesiones | 6/6 | 🟢 🟡 🔴 |
| Progresión en compuestos | ≥70% | 🟢 🟡 🔴 |
| Fatiga percibida | ≤7/10 | 🟢 🟡 🔴 |
| Dolor articular | 0–2/10 | 🟢 🟡 🔴 |
| Energía general | ≥7/10 | 🟢 🟡 🔴 |

### Interpretación del Semáforo

- 🟢 **Verde** = Todo dentro de objetivo → Continúa
- 🟡 **Amarillo** = Alerta leve → Monitorea
- 🔴 **Rojo** = Fuera de objetivo → Toma acción

## 3. Métricas de Progresión

| Métrica | Fórmula | Objetivo |
|---------|---------|----------|
| % ejercicios que progresan | (ejercicios que mejoran / total) | **≥70%** |
| Progresión en compuestos | Conteo semanal | **≥4/6** |
| Progresión en aislamientos | Conteo | **≥60%** |

### Interpretación

- Si % ejercicios que progresan **cae 2 semanas seguidas** → estás entrando en fatiga acumulada
- Acción: Anticipar deload o reducir volumen

---

## 4. Fatiga y Recuperación

Evalúa al final de cada día de entrenamiento:

| Indicador | Escala 1–10 | Comentario |
|-----------|-------------|-----------|
| Fatiga general | | Percepción subjetiva |
| Energía | | Disponible para tareas |
| Calidad de sueño | | Horas + profundidad |
| Estrés | | Laboral + personal |

### Interpretación

- **Fatiga > 8** → Riesgo alto de overtraining
- **Energía < 6** → Rendimiento comprometido
- **Sueño < 6 horas** → Recuperación insuficiente

## 5. Salud Articular (CRÍTICO)

Monitorea semanalmente por zona:

| Zona | Dolor (0–10) | Acción |
|------|--------------|--------|
| Hombros | | |
| Codos | | |
| Rodillas | | |
| Lumbar | | |

### Reglas

- **Dolor ≥5** → Ajustar volumen inmediato
  - Reducir sets en ejercicios que comprometan esa zona
  - Cambiar variante si es necesario
  
- **Persistente 2 semanas** → Modificar ejercicios
  - Buscar alternativa sin dolor
  - Consultar profesional si persiste

## 8. Resumen Semanal (Decisión Estratégica)

### Preguntas a Responder (Fin de semana)

- ✔️ ¿Progresé en los compuestos?
- ✔️ ¿Me siento recuperado?
- ✔️ ¿Hay dolor acumulado?

### Tabla de Decisión

| Condición | Acción |
|-----------|--------|
| Todo bien | ✓ Seguir progresión |
| Fatiga alta (>8/10) | Reducir aislamientos (max 2 ejercicios) |
| Dolor articular (≥5/10) | Ajustar ejercicios problema |
| Estancamiento 2+ semanas | Aplicar deload antes de lo previsto |
| Progresión fuerte | Continuar, monitorear fatiga |

---

## 9. Interpretación Tipo Negocio (Esto es lo Clave)

Piénsalo como un negocio:

```
Progresión = Revenue (ingresos)
Fatiga = Costos (gastos operativos)
Dolor = Riesgo operativo (puede quebrar empresa)
```

**Tu objetivo:** Maximizar revenue (progresión) sin quebrar la empresa (lesiones).

## Recomendaciones Finales

1. **Registra datos DURANTE el entrenamiento** (no después)
2. **Revisa semáforo cada viernes** para ajustar próxima semana si es necesario
3. **Compara 2 semanas atrás**, no solo la anterior (variabilidad normal)
4. **Si dos semanas rojas seguidas** → Evalúa deload anticipado
5. **Prioriza compuestos > todo lo demás**
